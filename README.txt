# ====================================================================================================================================================
# To-Do List con JavaScript 20-oct-2024
# ====================================================================================================================================================

## Descripción
# ====================================================================================================================================================

Este proyecto es una aplicación de lista de tareas desarrollada utilizando HTML, CSS, y JavaScript. Permite a los usuarios crear tareas, 
visualizarlas en un dashboard, y moverlas entre diferentes estados (pendientes, en progreso, y completadas) mediante funcionalidades de drag and drop.

## Requisitos Previos
# ====================================================================================================================================================

Para ejecutar este proyecto, necesitarás:
- Un navegador web moderno que soporte HTML5, CSS3, y JavaScript ES6.
- Acceso a Internet para cargar las librerías de Bootstrap y jQuery desde CDN.

## Instalación
# ====================================================================================================================================================

1. Clonar repositorio en la máquina local usando `git clone URL_DEL_REPOSITORIO`.
2. Abrir el archivo `index.html` en tu navegador para acceder a la aplicación.

## Estructura de Archivos
# ====================================================================================================================================================

- **index.html**: Página principal que sirve como punto de entrada al sitio web.
- **/assets/images/logo.webp**: Logo utilizado en la barra de navegación y pestañas del navegador.
- **/css/Styles.css**: Hoja de estilos que personaliza la apariencia por encima de Bootstrap.
- **/interfaces/interfaz1.html**: Dashboard que muestra tareas guardadas.
- **/interfaces/interfaz2.html**: Interfaz para la creación de nuevas tareas.
- **/interfaces/interfaz3.html**: Permite mover tareas entre diferentes estados.
- **/js/script.js**: Contiene la lógica para manejar tareas y su almacenamiento en LocalStorage.
- **/docs**: Documentos con información básica de HTML, CSS, Bootstrap, y JavaScript. Contiene además un informe explicativo de la realización del
    producto.

## Uso
# ====================================================================================================================================================

- **Dashboard (interfaz1.html)**: Visualiza las tareas guardadas y navega a otras interfaces.
- **Crear Tarea (interfaz2.html)**: Crea nuevas tareas y guárdalas.
- **Gestión de Tareas (interfaz3.html)**: Mueve las tareas entre los estados pendientes, en progreso y completadas usando drag and drop.
